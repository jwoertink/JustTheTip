Just The Tip jQuery Tool Tip

What is it?

A jQuery Tool Tip plug-in. Throw some HTML into a container, and have it appear next to the mouse cursor when you mouse over another element.


Why another tooltip?

Well, there's many different types of tooltips. Some are for input elements, some are for links. To me, it's a way to display information when I mouse over an element. I wanted something short and simple, so this is my go at it!
